# Brewhaus

A brewery discovery app built with Vue 3 and the [Open Brewery DB API](https://www.openbrewerydb.org/).

## Setup

Requires Node 20+.

```
npm install
npm run dev
```

## Features

- Browse breweries with pagination
- Search by name
- Filter by brewery type
- Brewery detail page with address, phone, website, and map link
- Favorite breweries (saved to localStorage)
